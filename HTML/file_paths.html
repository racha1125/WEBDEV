Absolute File Path

Relative to the root directory

Example: C:/Users/Ras/Desktop/Project/Images/cat.png

Relative File Path

Relative to the current directory.

If you are working on index.html file in Project Directory.
The Project Directory is the current directory.

  • Example-1: Images/cat.png
  • Example-2: ./Images/cat.png
    "./" tells the system to stay in the current directory and search for file "cat.png"

If we want to go to Desktop Folder which contains our Project Folder
and get some image named "dog.png" from their using relative file path,
the path is as follows:

File Path: ../dog.png

the ""../"" takes us a level up in the file hierarchy.(i.e.,desktop folder which the contains the project folder)
